Packages required:
Numpy
Pandas
Scikitlearn

Data csv file of Mera Naulek glacier is already processed for model training. Data are in their orignal units. Data pre-processing like normalization is excluded in this project. 

######### Making Predictions #######

open linearreg.py and run the script. 
Terminal will show few input commands for the meteorological varibles. 
Input are already converted into float data type, so decimal value is supported for input.
The resulting prediction for snow depth is printed in the console.

####### NOTE ######
please donot rename the python script file and the data file.
